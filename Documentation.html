create : Object

Terminal.create {}

Object Contains Methods To Create New Terminal View Lines, Inside The Lines Array

Methods:

Terminal.create.terminal (The Parent Terminal)

Terminal.create.Line(text, color, style) (Print New Line To The Terminal)

Terminal.create.ELine(style) (Print New Empity Line To The Terminal)

Terminal.create.HLine(style) (Print New Horizontal Line To The Terminal, The Same As [hr] Tag)


text: Is the message or the string to display.

color: Is the css color of the text.

style: Is the css code to style the text.

MainTerminal : Object

MainTerminal is just a variable used by the Terminal Tools To Select And Control The Terminal Objects
So, Create One Method For All The Terminals Objects

Tools Methods Can Called From The Terminal UI

So Tools Methods Make It Easy On The Users


terminal.create.Line('Hello World!', Color.Pink, 'font-style: 25px; font-weight:bold;') [Terminal Statment]

log('Hello World!', Color.Pink, 'font-style: 25px; font-weight:bold;') [Shortcut Statment]
